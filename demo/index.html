What does this plugin do?
It adds faq list (collapse/expand) functionality + stores current collapsed/expanded states in a cookie
How do I start using it?
Download the .zip below, view this example (in folder "demo") and read the comments in togglr.js.
Can I customize the CSS?
Yes, you can apply any CSS styles to each element.
Which browsers are supported?
IE8+, Chrome, Firefox, Opera, Safari + Mobile Browsers
